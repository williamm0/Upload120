How it works

Metadata changes, not frame interpolation.

mvhd

The movie header stores global movie timing. Upload120 patches the movie timescale and duration fields.

mdhd

The media header stores per-track timing. Upload120 patches each track's media timescale and duration fields.

stts

The sample timing table is inspected to estimate FPS from video sample deltas and the video track timescale.

Auto mode does not multiply. It preserves the detected input FPS as the output/effective FPS. Choose 2×, 4×, or Custom when you intentionally want a higher apparent FPS.
