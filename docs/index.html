How it works

Only the timing metadata changes.

Upload120 inspects the MP4 structure, estimates video FPS from sample timing, and rewrites movie and media timing headers. It does not create new frames or alter image quality.

mvhd

Movie timing

Updates the global timescale and duration fields used by the container.

mdhd

Track timing

Updates timing fields on each media track so the file stays internally consistent.

stts

FPS estimate

Reads sample deltas to estimate the source frame rate before patching.
