About

Local-only desktop companion for Upload120.

Upload120 patches timing metadata and can post through TikTok's desktop web pipeline without sending files anywhere else.

Upload120

Version
Made by jx
mvhd

Movie timing

Updates the global timescale and duration fields used by the container.

mdhd

Track timing

Updates timing fields on each media track so the file stays internally consistent.

stts

FPS estimate

Reads sample deltas to estimate the source frame rate before patching.

Upload120 rewrites two metadata fields inside an MP4, mvhd and mdhd, so the upload pipeline reads the timeline as if the source were standard 30 fps. Your real frames stay untouched.

For best results, use a fixed-rate MP4 at 60 or 120 fps and upload the patched file from a computer. Phone uploads re-encode and undo the trick.

Everything runs locally on your computer. No uploads, no telemetry, no data leaves the app.
